When programming, it is useful to be able to group data into structures for later processing. Sour lang provides arrys for exactly this purpose.

Lists store items in the order that they are added, and allow for duplicate items.

to create array put all elements inside of [].

When creating lists, Sour lang can infer the type of items stored. To declare the type explicitly, add the type after : after the array declaration:

Arrays 
        const shapes = ["triangle", "square", "circle"]
        const fruits: str[] = ["apple", "banana", "mango"]
        const colors = ["red", "green", "blue"]: str
        
        print(shapes)
        print(fruits)
        print(colors)
