These are the following data types in Sour.

  • Numbers

Numbers

  • Positive integers 64, +64
  • Negative integers -64
  • Decimanls 4.5
Numbers 
          var a = 1
          print(a) // 1

Strings

  • Anything surrounded by double quotes "abc 929 ;,"
  • Use \" to a " between string "abc 929 \" ;," "abc " abc "
Strings 
          var str = "My name is \"Jony\"."
          
          print(str) // My name is "Jony".

Null

The idea of null is generally used to indicate the absence of a value or an empty reference. It's useful in situations where a variable might not have any data assigned to it yet but will assigned one in future.

Null Value 
          var name = null
          print(name) // null
          name = "Marky"
          print(name) // Marky
null value can be printed with print() function.
