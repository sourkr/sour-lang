Variables

All programs need to be able to store data, and variables help you to do just that. In Sour, you can declare:

  • read-only variables with const
  • mutable variables with var

To assign a value, use the assignment operator =.

For example: Variables 
        const popcorn = 5    // there are 5 boxes of popcorn
        const hotdog = 7     // There are 7 hotdogs
        var customers = 10   // There are 10 customers in the queue
        
        // Some customers leave the queue
        customers = 8
        print(customers) // 8

A variable can store any type of value and can be changed to another type of value.
