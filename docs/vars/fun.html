You can declare your own functions in Sour using the fun keyword.

fun hello() {
  print("Hello, World!")
  return "success"
}

hello()

In Sour Lang:

  • function parameters are written within parentheses ().
  • multiple parameters must be separated by commas ,.
  • the body of a function is written within curly braces {}.
  • the return keyword is used to exit or return something from a function.
In the following example:
  • x and y are function parameters.
  • the function returns a sum of x and y when called.
fun sum(x, y) {
  return x + y
}

print(sum(1, 2)) // 3

Assign to Variable

fun to_upper_case_str(str) {
  return str.uppercase()
}

var upper_case = to_upper_case_str

print(upper_case("Hello")) // HELLO

Pass to another function

fun positive(n) {
  return n > 0
}

var a = [ 0, 4, -2, 1, 5 ]

print(a.filter(positive)) // [ 4, 1, 5 ]
