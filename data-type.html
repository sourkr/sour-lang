Understanding Data Types in Sour

Data types define the kind of information a variable can hold in your Sour programs. Knowing data types is crucial for writing efficient and accurate Sour code.

Basic Data Types

  • **Numbers (Numeric):** Represent numerical values. Examples: `10`, `3.14`, `-25`.
  • **Strings:** Represent text data enclosed in double quotes. Examples: `"Hello, World!"`, `"This is a string."`.
  • **Booleans:** Represent logical values (true or false). Examples: `true`, `false`.

Using Data Types

Here's how you declare variables with different data types in Sour:

var age = 30;  // Integer (number)
var name = "Alice";  // String
var isLoggedIn = true;  // Boolean

You can use these variables in your Sour programs to perform calculations, store text, and make decisions based on conditions.

Advanced Data Types (Optional)

Sour may also support more complex data types depending on its features. Here are some examples found in many programming languages (check your Sour documentation for specifics):

  • **Arrays:** Ordered collections of elements of the same or different data types.
  • **Objects:** Collections of key-value pairs, where keys are unique identifiers and values can be of any data type.
